SEI!dat para um rápido acesso aos dados!

Extraia com facilidade os resultados das pesquisas realizadas no SEI!

SEI!dat permite a extração estruturada de metadados do SEI! para gerar relatórios.

Sobre o Projeto

Público-alvo

Servidores públicos que precisam extrair dados do SEI! para executar tarefas e gerar relatórios.

Qual o problema?

Resultado de pesquisa e relatórios do SEI! não permitem exportação de dados e não apresenta display amigável.

Objetivo

Desenvolver plugin que ofereça melhor recurso de visualização dos resultados de pesquisa e também a extração de dados do SEI!

Indexação e coleta de dados de dados baseada no uso de metadados dos documentos e processos
